El codigo en windows debe ser ejecutado como >type input.bat | python3 .\main.py
donde input.bat corresponde al input solicitado en la tarea

En Bash debe ser ejecutado como >cat input.bat | python3 .\main.py
donde de la misma forma, input.bat contiene el input solicitado por la tarea

El input es de tipo

3
6 ABCLGH
6 AELFHR
6 AGGJAB
7 GZJZAMB
16 Este es un texto
18 Este es otro texto
